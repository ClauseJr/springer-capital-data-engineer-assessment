Referral Reward Validation Pipeline

Springer Capital Data Engineering Assessment A complete end-to-end data engineering solution developed in Python to profile, clean, integrate and validate referral reward data using business rules while detecting potential inconsistencies and fraud.

View GitHub Repository

Project Overview

Objective

Build a complete referral reward validation pipeline capable of profiling multiple datasets, joining relational tables and determining whether referral rewards satisfy business rules.

Business Problem

Determine valid and invalid referral rewards based on referral status, transaction history, membership validity, reward issuance and referral activity.

Output

Generate a clean CSV report containing every referral together with its validation status for business stakeholders.
